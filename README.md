# 💊 Farmácia VAV - Gerenciamento de Estoque
![PHP](https://img.shields.io/badge/PHP-8.x-blue)
![MySQL](https://img.shields.io/badge/MySQL-Database-orange)

Sistema web de gerenciamento de estoque para uma farmácia fictícia (Farmácia VAV). Este projeto foi desenvolvido como atividade prática com foco na construção de um **CRUD completo em PHP utilizando PDO**, garantindo segurança e boas práticas.

## 🎯 Objetivos do Projeto

- **Back-end:** Implementar um sistema de inserção, leitura, atualização e exclusão (CRUD) seguro utilizando PHP PDO (Prepared Statements).
- **Front-end:** Criar uma interface responsiva aplicando a metodologia **Mobile First** e a **Regra de Cores 60-30-10**.
- **Colaboração:** Utilizar o Git e GitHub para versionamento de código em equipe.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Semântica estrutural.
- **CSS3:** Estilização baseada em Mobile First.
- **PHP:** Lógica de negócio e comunicação com o banco.
- **PDO (PHP Data Objects):** Conexão e manipulação segura de dados.
- **MySQL:** Banco de dados relacional.

## 🎨 Design (Regra 60-30-10)

O layout foi planejado seguindo a proporção de cores:
- **60% (Fundo/Neutro):** Tons claros para facilitar a leitura.
- **30% (Secundária/Saúde):** Tons de Azul/Verde nas barras de navegação e menus.
- **10% (Destaque/Ação):** Tons de Laranja/Vermelho para botões de exclusão e alertas.

## 🚀 Como Executar o Projeto Localmente

### Pré-requisitos
Você precisará de um servidor local como [XAMPP](https://www.apachefriends.org/), [WAMP](https://www.wampserver.com/) ou [Laragon](https://laragon.org/) para rodar o PHP e o MySQL.
